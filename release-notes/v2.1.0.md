# Mom Jeans — sync input fix

The **SYNC input** now triggers cleanly on the rising edge. Previously it used a
level check that retriggered the oscillator phase on *every sample* the sync
signal stayed high, producing noise. This affected both the MetaModule and VCV
Rack builds.

This release contains the same fix for both platforms, so a patch behaves the
same whether you run it on MetaModule hardware or in VCV Rack.

## Downloads

- **MetaModule:** `CuteLab-v2.1.0.mmplugin` — requires firmware **v2.2.0 or
  newer**.
- **VCV Rack:** `CuteLab-2.1.0-*.vcvplugin` — pick your platform (mac-arm64,
  mac-x64, win-x64, lin-x64).

## Note for VCV Rack users

This is an **unofficial build** published here for direct download. The official
CuteLab plugin in the VCV Rack Library is maintained separately, by CuteLab, and does not yet include this fix. Install it by hand if you want the sync fix in VCV Rack, and note that it will replace the library version.
